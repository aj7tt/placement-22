## Common Coding Interview Questions 

**Q #1) How can you reverse a string?**

~ void reverseStr(string& str)
{
    int n = str.length();
 
    // Swap character starting from two corners
    for (int i = 0; i < n / 2; i++)
        swap(str[i], str[n - i - 1]);
}
